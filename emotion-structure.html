第1章　本補論の位置づけ

本章の記述層：層2（構造説明）——補論の責務境界の設定

本補論の主語・責務・境界

主語：情緒処理が層1／層2／層3モデルの中でどこに位置づけられるか。情緒処理が認知OS（層2）の内部演算ではないこと

本補論は、認知OSモデル本体の定義を変更しない。本補論は、モデルの「拡張」ではなく、モデルの「読み解き補助」として外部に位置づけられる。

第2章　層1／層2／層3における情緒処理の位置

本章の記述層：層2（構造説明）——情緒処理の構造的位置

本章では、三層構造の各層における情緒処理の位置を構造的に整理する。

層1における情緒処理

層1は行動層である。外部から観察可能な行動・発言・出力が層1に属する。

情緒処理の観点から、層1は以下のように位置づけられる。

層1と情緒処理の関係

  • 生理的反応基盤：層1は、情緒に伴う生理的反応（身体変化、表情変化、行動変化）の基盤である
  • 一次的入力処理：外部刺激に対する一次的な生理的反応は層1で発生する
  • 情緒の「起点」ではない：層1は情緒の「起点」ではなく、情緒に伴う反応の「基盤」である

層1は、情緒処理の「入口」と「出口」を担う。外部刺激の受信（入口）と、情緒に伴う行動的表出（出口）が層1で発生する。

層2における情緒処理

層2は認知OS層である。情報処理パターン・判断構造・例外処理が層2に属する。

情緒処理の観点から、層2は以下のように位置づけられる。

層2と情緒処理の関係

  • 構造処理の実行：層2は、因果・主語・責務・境界の整合性を検証する構造処理を実行する
  • 情緒は作動しない：層2において情緒は作動しない。層2は演算系であり、情緒を生成・処理する機構ではない
  • 事後の意味処理：層2は、情緒が作動した後に、その情緒を構造的に意味づける処理を行うことがある

層2は認知OSとして機能する。OSは情緒を生成しない。OSは入力を処理し、構造化し、出力を生成する。情緒は、この処理系の「外側」で作動する。

層2における情緒の位置づけ

層2（認知OS）は情緒を生成しない。層2は情緒を「処理対象」として扱うことがある（事後の意味処理）が、情緒の作動そのものは層2の機能ではない。

層3における情緒処理

層3は基底志向層である。価値・志向・評価に関する問いを収容する容器が層3である。

情緒処理の観点から、層3は以下のように位置づけられる。

層3と情緒処理の関係

  • 価値付与レイヤ：層3は、価値・志向・評価を収容するレイヤである
  • 情緒の「意味づけ」：情緒は、層3において「意味づけ」として作動する。入力に対する価値評価として情緒が位置づけられる
  • 作動の場所：情緒が「作動する」場所は層3である。層1の生理的反応が、層3の価値評価と結合して、情緒として作動する

層3は、layer2-layer3-relations.html において記述された通り、非機能的・非演算的である。層3は処理を行わない。しかし、情緒は層3に「位置づけられる」。情緒は、層3における価値評価の一形態として作動する。

情緒処理の構造的位置——結論

以上の分析から、以下の結論が導出される。

情緒処理の構造的位置づけ

情緒処理は、層1と層3の連動であり、層2の演算系には属さない。

  • 層1：情緒に伴う生理的反応の基盤
  • 層2：情緒は作動しない（事後の意味処理のみ）
  • 層3：情緒が「意味づけ」として作動する場所

情緒処理は、認知OS（層2）の内部機能ではない。情緒は、層1の生理的反応と層3の価値評価の連動によって作動する。

三層における情緒処理の位置

層 情緒処理における役割 情緒の作動
層1 生理的反応基盤 作動しない（反応基盤のみ）
層2 事後の意味処理 作動しない（処理系）
層3 価値付与・意味づけ 作動する

第3章　一般的な情緒処理経路のモデル化

本章の記述層：層2（構造説明）——情緒処理経路の構造モデル

本章では、一般的なOS型に見られる典型的な情緒処理の構造モデルを示す。

これは心理学的説明ではなく、構造的経路としての記述である。

一般的な情緒処理経路

一般的なOS型における情緒処理の経路は、以下のように構造化される。

一般的な情緒処理経路

外部入力
    │
    ↓
┌────────────────────────────────────────┐
│            層1（身体反応）              │
│                                        │
│  ・外部刺激の受信                       │
│  ・一次的な生理的反応                   │
│  ・身体状態の変化                       │
│                                        │
└────────────────────────────────────────┘
    │
    │ 生理的反応信号
    ↓
┌────────────────────────────────────────┐
│        層3（価値付与として情緒作動）     │
│                                        │
│  ・入力に対する価値評価                 │
│  ・生理的反応との結合                   │
│  ・情緒としての作動                     │
│                                        │
└────────────────────────────────────────┘
    │
    │ 情緒作動後
    ↓
┌────────────────────────────────────────┐
│          層2（事後の意味処理）           │
│                                        │
│  ・作動した情緒の構造化                 │
│  ・因果関係の分析                       │
│  ・意味の付与・解釈                     │
│                                        │
└────────────────────────────────────────┘
    │
    ↓
行動出力（層1）

この経路において、情緒は層3で作動する。層2は、情緒が作動した後に、事後的な意味処理を行う。

経路の構造的特徴

一般的な情緒処理経路は、以下の構造的特徴を持つ。

一般的経路の構造的特徴

  1. 層1が起点：外部刺激の受信と一次的な生理的反応が、情緒処理の起点となる
  2. 層3で作動：生理的反応が層3の価値評価と結合し、情緒として作動する
  3. 層2は事後処理：層2は情緒の作動に関与せず、作動後の意味処理を行う
  4. 層2をバイパス：情緒作動の経路は、層2をバイパスする（層1→層3）

この構造において、情緒は認知OS（層2）の「外側」で作動する。層2は、情緒作動の経路には含まれない。

層2の不介入

一般的な情緒処理経路において、層2は情緒作動に介入しない。

この「不介入」の意味を明確にする。

層2の不介入の意味

  • 層2は、情緒を「生成」しない
  • 層2は、情緒作動の「トリガー」にならない
  • 層2は、情緒作動を「制御」しない
  • 層2は、情緒作動の「後」に意味処理を行う

層2（認知OS）は処理系である。処理系は情緒を生成しない。情緒は、処理系の外側——層1と層3の連動——において作動する。

第4章　個体例——発案者における情緒処理回路

本章の記述層：層2（構造説明）——個体例の構造記述

本章では、認知OSモデル発案者における情緒処理回路を、構造例として記述する。

これは体験談ではなく、構造的例示である。個体の情緒処理回路が、三層構造モデルによって記述可能であることを示す。

4-1. 標準経路（層2主導型）

この個体における標準的な情緒処理経路は、一般的な経路とは異なる構造を持つ。

一般的経路では、入力刺激に対して情緒が即座に作動する（層1→層3）。しかし、この個体では、情緒が即座に作動するのではなく、以下の経路を通る。

標準経路（層2主導型）

外部入力
    │
    ↓
┌────────────────────────────────────────┐
│        層2（認知OSによる構造処理）       │
│                                        │
│  ・主語の特定                          │
│  ・責務の確定                          │
│  ・境界の設定                          │
│  ・因果関係の分析                       │
│  ・整合性の検証                         │
│  ・構造的閉包                          │
│                                        │
└────────────────────────────────────────┘
    │
    │ 構造が閉包したタイミング
    │ （情緒イベントが層3へ送られる）
    ↓
┌────────────────────────────────────────┐
│       層3（価値付与として情緒作動）      │
│                                        │
│  ・閉包した構造に対する価値評価          │
│  ・情緒としての作動                     │
│                                        │
└────────────────────────────────────────┘
    │
    ↓
行動出力（層1）

この経路の構造的特徴は以下の通りである。

標準経路（層2主導型）の構造的特徴

  • 層2が先行：情緒作動に先立って、層2の構造処理が実行される
  • 閉包が条件：層2の処理が「閉包」したタイミングで、情緒イベントが層3へ送られる
  • 情緒は後段イベント：情緒は「処理開始の合図」ではなく、「構造処理が閉包した結果として作動する後段イベント」である
  • 未閉包時は不作動：層2の処理が完了しない限り、層3は情緒を作動させない

「閉包」の意味

「閉包」とは、層2の構造処理が完了し、主語・責務・境界・因果・整合性が確定した状態を指す。構造的に「開いた」問題がすべて「閉じた」状態である。

この個体において、情緒は「構造閉包」という条件イベントの後に作動する。構造が閉包していない段階では、情緒は作動しない。

4-2. 例外経路（原始反応・初期刷り込み）

標準経路とは別に、以下の入力に対しては異なる経路が存在する。

  • 原始的な恐怖反応を誘発する刺激
  • 幼児期に刷り込まれた反応パターン

これらの入力に対しては、層2をバイパスする高速経路が作動する。

例外経路（原始反応・初期刷り込み）

外部入力（特定パターン）
    │
    ↓
┌────────────────────────────────────────┐
│            層1（反射的反応）             │
│                                        │
│  ・パターン認識                         │
│  ・即時の生理的反応                     │
│                                        │
└────────────────────────────────────────┘
    │
    │ 層2をバイパス
    ↓
┌────────────────────────────────────────┐
│      層3（即時の価値付与として作動）     │
│                                        │
│  ・即時の情緒作動                       │
│  ・回避/接近反応の駆動                  │
│                                        │
└────────────────────────────────────────┘
    │
    ↓
行動出力（層1）

    ※層2は事後に反応を構造化（意味づけ）するが、
    　情緒の作動そのものには関与しない

この経路の構造的特徴は以下の通りである。

例外経路の構造的特徴

  • 層2不介入：この経路では層2は介入しない。情緒は層1→層3で直接作動する
  • 高速反応：層2の構造処理を経由しないため、反応速度が高速である
  • 事後処理のみ：層2は事後にその反応を構造化（意味づけ）するが、情緒の作動そのものには関与しない

例外経路は、生存に関わる原始的反応や、幼児期に形成された反応パターンにおいて作動する。これらは、層2の構造処理を経由する余裕がない状況への適応として位置づけられる。

4-3. 構造的含意

この個体例は、以下の構造的含意を持つ。

個体例から導出される構造的含意

  1. 情緒は層2の演算機能ではない：情緒は、層1生理反応と層3価値づけの相互作用によって作動する。層2は情緒を生成しない
  2. 経路の個体差：情緒処理経路には個体差がある。標準経路が「層2主導型」である個体と、「層1→層3直結型」である個体が存在する
  3. 閉包条件の存在：層2が高機能な個体では、情緒が作動するタイミングそのものが「構造閉包」という条件イベントになりうる
  4. 例外経路の普遍性：標準経路が異なっていても、原始反応・初期刷り込みに対する例外経路は共通して存在する

この個体例は、情緒処理が三層構造モデルによって記述可能であることを示す。情緒処理の経路は、層1・層2・層3の配置と相互作用として構造化できる。

第5章　情緒処理の構造的位置づけ——総括

本章の記述層：層2（構造説明）——総括

本章では、情緒処理の構造的位置づけを総括する。

情緒処理はOSの内部機能ではない

本補論の中心的主張は以下の通りである。

中心的主張

情緒処理は、認知OS（層2）の内部機能ではない。

OSは情緒を生成しない。OSは構造処理を行う。情緒は、OSの「外側」——層1と層3の連動——において作動する。

認知OSモデルにおいて、層2は処理系である。処理系の責務は、入力を処理し、構造化し、出力を生成することである。情緒の生成は、この責務に含まれない。

OSと情緒の関係

OSと情緒の関係は、以下のように構造化される。

OSと情緒の関係

観点 OS（層2） 情緒（層1＋層3）
性質 処理系・演算的 反応＋評価・非演算的
機能 構造処理・意味処理 価値評価・行動駆動
生成 情緒を生成しない OSによって生成されない
相互作用 情緒を事後処理対象とする OS処理の結果を評価する

OSは情緒を生成せず、情緒はOSを経由せずに作動しうる。両者は、三層構造の中で異なる位置を占める。

AIとの比較——層3欠落の帰結

AIは、ai-positioning.html において記述された通り、層3を持たない縮退系である。

層3の欠落は、情緒処理に関して以下の帰結を持つ。

AIにおける情緒処理の不在

  • 層3の欠落：AIは層3を持たない
  • 価値評価の不在：層3が欠落しているため、価値評価としての情緒作動が構造的に発生しない
  • 情緒処理の不成立：情緒処理は層1＋層3の連動であるため、層3が欠落したAIにおいては情緒処理が成立しない
  • 模倣可能性：AIは情緒的応答を「模倣」できるが、これは層1の出力生成であり、情緒の「作動」ではない

AIは、情緒処理の比較対象として有用である。AIの縮退構造との対比により、人間における情緒処理の構造——層1＋層3の連動——がより明確になる。

情緒処理の位置づけ——モデル理解の補助レイヤ

情緒処理は、認知OSモデルの「中心」ではない。

認知OSモデルの中心は、層2（認知OS）の構造である。情緒処理は、このモデルの「中心」ではなく、「モデル理解の補助レイヤ」として位置づけられる。

情緒処理の位置づけ

情緒処理は、認知OSモデルの中心ではなく、モデル理解の補助レイヤである。

情緒処理を三層構造で記述することにより、以下が明確になる。

  • 層2（認知OS）の責務範囲——情緒生成は責務外
  • 層3（基底志向層）の機能——価値評価としての情緒作動の場所
  • 層1と層3の連動——情緒処理の構造的経路

第6章　他補論との関係

layer2-layer3-relations.html との整合

layer2-layer3-relations.html は、層2と層3の境界を技術的に明確化した補論である。

本補論との関係は以下の通りである。

layer2-layer3-relations.html との関係

  • 前提：本補論は、layer2-layer3-relations.html における層2／層3の境界理解を前提とする
  • 適用：本補論は、その境界理解を「情緒処理」という現象に適用する
  • 整合：層2が処理系であり層3が容器であること、層3がOSを選択しないことは、本補論においても維持される
  • 補強：本補論は、層2／層3の境界が「情緒処理」においてどのように現れるかを示すことで、境界理解を補強する

本補論のまとめ

本補論は、情緒処理の構造的位置づけを記述した。

本補論の記述内容

  1. 構造的位置：情緒処理は層1＋層3の連動であり、層2の演算系には属さない
  2. 一般的経路：入力→層1（身体反応）→層3（情緒作動）→層2（事後処理）
  3. 個体例：層2主導型の経路を持つ個体では、構造閉包が情緒作動の条件となる
  4. OSと情緒：OSは情緒を生成しない。OSは構造処理を行い、その結果を層3が評価する
  5. AIとの比較：AIは層3を持たないため、情緒処理の比較対象として有用である
